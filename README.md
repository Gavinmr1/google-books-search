# Google Books Search

https://fast-taiga-11126.herokuapp.com/

## Description

This is a new React-based Google Books Search app. This app was created using React components, helper/util functions, and utilizes React lifecycle methods to query and display books based on user searches. It also uses Node, Express and MongoDB so that users can save books to review or purchase later.

  ## Installation
  
  *Steps required to install project and how to get it running:*
  
  First the user must run 'npm install'. Then install dependencies. After the intalls are complete the user may run 'node server.js' to start the application.
